# Titanic Survival Prediction

This is a practice machine learning project built using the Titanic dataset.

The goal of this project is to predict whether a passenger survived the Titanic disaster using classification models.

### Steps performed:

- Data cleaning
- Exploratory Data Analysis
- Feature engineering
- Model training
- Model evaluation
- Building a prediction model

### Tools used:

Python, NumPy, Pandas, Matplotlib, Seaborn, Scikit-learn
